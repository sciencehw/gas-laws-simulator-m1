# gas-laws-simulator-m1
# ⚗️ 기체 법칙 시뮬레이션 웹앱

중학교 1학년 과학 교육을 위한 인터랙티브 시뮬레이션 웹 애플리케이션입니다.

## 📚 교육 목표

**2015 개정 교육과정 성취기준**
- [9과06-02] 기체의 압력과 부피 관계를 실험 결과로부터 알아내고, 이를 입자 모형으로 해석할 수 있다.
- [9과06-03] 기체의 온도와 부피 관계를 실험 결과로부터 알아내고, 이를 입자 모형으로 해석할 수 있다.

## 🎯 주요 기능

### 1. 보일의 법칙 시뮬레이션
- 압력과 부피의 반비례 관계 학습
- 실시간 입자 모형 애니메이션
- 슬라이더를 통한 압력 조절 (0.5~3.0 기압, 0.1 기압 단위)
- 측정 데이터 자동 기록 및 테이블 표시
- 엑셀(CSV) 파일 다운로드 기능

### 2. 샤를의 법칙 시뮬레이션
- 온도와 부피의 비례 관계 학습
- 온도에 따른 입자 운동 변화 시각화
- 슬라이더를 통한 온도 조절 (0~100°C, 1°C 단위)
- 측정 데이터 자동 기록 및 테이블 표시
- 엑셀(CSV) 파일 다운로드 기능

### 3. 형성평가
- 15개 문제 중 랜덤 5문제 제공
- 즉각적인 피드백 제공
- 정답/오답 확인 기능

### 4. 추가 기능
- 학생 정보 자동 입력 및 파일명 반영
- 개념 설명 및 실생활 예시 제공
- 탭별 랜덤 퀴즈 (각 2문제)
- 데이터 초기화 기능
- 모바일 반응형 디자인

## 🚀 GitHub Pages 배포 방법

### 1. 저장소 생성
```bash
# 새 저장소 생성
mkdir gas-laws-simulator
cd gas-laws-simulator
git init
```

### 2. 파일 추가
```bash
# 모든 파일을 프로젝트 폴더에 복사 후
git add .
git commit -m "Initial commit: 기체 법칙 시뮬레이션 웹앱"
```

### 3. GitHub에 푸시
```bash
# GitHub에서 새 저장소 생성 후
git remote add origin https://github.com/사용자명/gas-laws-simulator.git
git branch -M main
git push -u origin main
```

### 4. GitHub Pages 활성화
1. GitHub 저장소 페이지에서 **Settings** 클릭
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Source**에서 `main` 브랜치 선택
4. **Save** 클릭
5. 몇 분 후 `https://사용자명.github.io/gas-laws-simulator/` 에서 접속 가능

## 📱 사용 방법

### 학생용
1. 웹사이트 접속 시 학번과 이름 입력
2. 원하는 탭 선택 (보일의 법칙 / 샤를의 법칙 / 형성평가)
3. 슬라이더를 조작하여 변화 관찰
4. "데이터 기록" 버튼으로 측정값 저장
5. "엑셀 다운로드" 버튼으로 실험 결과 저장
6. 형성평가로 학습 내용 확인

### 교사용
- 학생들이 제출한 CSV 파일로 실험 결과 확인 가능
- 파일명에 학번과 이름이 자동 포함되어 관리 용이
- 형성평가 결과로 학생들의 이해도 파악 가능

## 🛠️ 기술 스택

- **HTML5**: 구조 및 마크업
- **CSS3**: 반응형 디자인 및 애니메이션

- **Canvas API**: 입자 애니메이션
